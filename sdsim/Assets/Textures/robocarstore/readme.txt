This scene contains the robocars standard track, with a random background and random track color options (disabled by default)

- Random background:
The random background option can be accessed through the RandomBackgroundManager object, there are 20 HDRIs included (all from https://hdrihaven.com/, CC0 license)

- Random track color
The random track color option can be accessed through RandomMatColor inside the track objects. The track are separated into different layers
